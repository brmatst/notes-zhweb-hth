# notes-zhweb-hth

本仓库用于归档和发布多个独立的 HTML 页面。这些页面来源于日常整理与记录，不针对特定域名或具体网站，仅供查阅和参考。

## 项目简介

这是一个以 HTML 页面形式存放笔记与存档的仓库。页面内容主要涉及技术笔记、知识整理或其他个人记录，以静态网页形式呈现，便于直接通过浏览器打开查看。仓库本身不提供动态功能，也不依赖后端服务。

## 目录说明

```
notes-zhweb-hth/
├── pages/          # 存放独立的 HTML 页面文件
├── assets/         # 页面引用的静态资源（如图片、样式、脚本）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，可单独打开浏览。
- `assets/`：存放页面所需的公共资源文件，按需引用。

## 页面归档说明

- 所有页面均为静态 HTML，不包含服务端逻辑。
- 页面内容可能随时更新或整理，不保证长期不变。
- 部分页面可能引用外部资源（如 CDN 上的库文件），打开时需保持网络连接。

## 维护说明

- 本仓库不定期更新，新增或修改页面。
- 不承诺对历史页面内容进行持续维护或修正。
- 若您发现页面存在明显错误，欢迎通过 Issue 或 Pull Request 提出建议。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `pages/` 目录下的 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务部署后在线访问（需自行配置）。
